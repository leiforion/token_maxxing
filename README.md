# Project Setup Guide

## 📋 Prerequisites & Downloads

Before getting started, please download and install the following tools and files:

* 📦 **[TSV File (zipped)](https://drive.google.com/file/d/1sDrhA6qW0REbUxMftavYAxgvLFXPyy77/view?usp=sharing)**
* 🐳 **[Docker](https://www.docker.com/get-started/)**: Click **'Download Docker Desktop'** on the page and follow the installation instructions.
* 💻 **[Cursor](https://cursor.com/get-started)**: Download and install the Cursor editor.

---

## 🚀 Setup Steps

1. **Prepare your project folder**
   Create a new folder on your computer.

2. **Download and extract the data**
   Download the [TSV File](https://drive.google.com/file/d/1sDrhA6qW0REbUxMftavYAxgvLFXPyy77/view?usp=sharing) into the folder you just created. 
   > ⚠️ **Important:** Unzip the file in this folder, but do **NOT** open the TSV file itself.

3. **Open the project in Cursor**
   Install Cursor if you haven't already. Once installed, open Cursor and open the folder you created (the one containing the unzipped TSV file).

4. **Save your workspace**
   In the Cursor editor, go to the menu and click **'Save as Workspace'**.
